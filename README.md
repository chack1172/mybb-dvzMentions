# DVZ Mentions

A MyBB 1.8.x plugin for parsing `@username`, `@username#id` mentions into profile links. Integrates with _MyAlerts_.

Compatible with JSON-encoded AJAX requests going through `xmlhttp.php`.

**Requirements:**
- PHP >= 7.0
